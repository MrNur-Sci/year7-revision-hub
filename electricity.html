1. Circuit Basics

Cell

Provides energy. It is drawn as one long line and one short line.

Current

Flow of charge. Measured in amps (A).

Resistance

How hard it is for current to flow. Measured in ohms (Ω).

4. Series vs Parallel Summary

FeatureSeriesParallel
PathwaysOne loopMore than one branch
CurrentSame through componentsSplits between branches
VoltageShared between componentsSame across each branch
ResistanceAdding components increases total resistanceAdding branches can reduce total resistance
If one part breaksWhole circuit stopsOther branches can still work

5. Static Electricity

Balloon test: attract paper or bend water

Charge level 0
Test object Paper
Rule
Rubbing can transfer charge. A charged object can attract neutral objects because charges inside them shift slightly. This can attract paper or bend a thin stream of water.
